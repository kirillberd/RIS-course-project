Добро пожаловать в ИС Билборды

О системе

Информационная система предназначена для управления рекламными щитами (билбордами) и включает следующие возможности:

Для клиентов:

  • Поиск доступных билбордов
  • Фильтрация по городу, размеру и стоимости
  • Добавление билбордов в корзину
  • Оформление заказов на аренду

Для владельцев билбордов:

  • Размещение новых билбордов
  • Управление существующими билбордами
  • Просмотр статистики аренды

Для аналитиков:

  • Доступ к базе данных пользователей
  • Создание и просмотр отчетов
  • Анализ статистики использования
